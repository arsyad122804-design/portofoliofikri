# Folder Books

Masukkan gambar cover buku yang sudah dibaca ke dalam folder ini.

## File yang diperlukan:
- `book1.jpg` - Cover buku pertama
- `book2.jpg` - Cover buku kedua
- `book3.jpg` - Cover buku ketiga
- `book4.jpg` - Cover buku keempat
- `book5.jpg` - Cover buku kelima
- `book6.jpg` - Cover buku keenam

## Spesifikasi:
- Format: JPG atau PNG
- Ukuran: 300x450px (portrait/vertikal)
- Resolusi: Minimal 72 DPI
- Ukuran file: Maksimal 500KB per gambar

## Tips:
- Gunakan cover buku asli untuk hasil terbaik
- Pastikan gambar tidak blur atau pecah
- Nama file harus sesuai dengan yang tertera di atas
- Anda bisa menambah lebih banyak buku dengan menambahkan `book7.jpg`, `book8.jpg`, dst.
