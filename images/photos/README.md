# Folder Photos (Foto Kenangan)

Masukkan foto-foto kenangan ke dalam folder ini.

## File yang diperlukan:
- `photo1.jpg` sampai `photo6.jpg` (atau lebih)

## Spesifikasi:
- Format: JPG atau PNG
- Ukuran: 800x800px (format persegi untuk hasil terbaik)
- Resolusi: Minimal 72 DPI
- Ukuran file: Maksimal 2MB per foto

## Tips:
- Gunakan foto dengan resolusi tinggi
- Format persegi akan tampil lebih rapi
- Bisa menambah lebih banyak foto dengan menambahkan photo7.jpg, photo8.jpg, dst.
- Edit di index.html untuk menambah/mengurangi foto
