# Folder Images

Masukkan gambar-gambar berikut ke dalam folder ini:

## File Gambar Utama:
- `profile.jpg` - Foto profil untuk hero section (300x300px, format persegi dengan background merah)
- `profile-full.jpg` - Foto profil lengkap untuk about section (400x600px, foto full body)
- `contact-photo.jpg` - Foto untuk contact section (200x200px, format bulat)
- `thank-you.jpg` - Gambar thank you (landscape, 800x600px)
- `portfolio-preview.jpg` - Preview portfolio (landscape, 600x400px)

## Folder icons/:
Masukkan icon software (50x50px, format PNG dengan background transparan):
- `figma.png` - Icon Figma
- `photoshop.png` - Icon Adobe Photoshop
- `lightroom.png` - Icon Adobe Lightroom
- `capcut.png` - Icon CapCut
- `elementor.png` - Icon Elementor
- `premiere.png` - Icon Adobe Premiere Pro

## Folder projects/:
Masukkan 8 foto project Anda (format persegi 500x500px):
- `project1.jpg` sampai `project8.jpg`

## Tips:
- Gunakan foto dengan resolusi tinggi untuk hasil terbaik
- Format yang disarankan: JPG untuk foto, PNG untuk icon
- Pastikan nama file sesuai dengan yang tertera di atas
