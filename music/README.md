# Folder Music

Masukkan file musik background ke dalam folder ini:

## File yang diperlukan:
- `background.mp3` - File musik background untuk website

## Spesifikasi:
- Format: MP3
- Durasi: 2-5 menit (akan di-loop otomatis)
- Ukuran file: Maksimal 5MB untuk loading yang cepat
- Bitrate: 128-192 kbps (recommended)

## Catatan:
- Musik akan otomatis play setelah user melakukan interaksi pertama (click)
- Musik akan di-loop secara otomatis
- Pilih musik yang tidak terlalu keras dan cocok untuk background
- Pastikan Anda memiliki hak untuk menggunakan musik tersebut

## Rekomendasi Genre:
- Lo-fi hip hop
- Ambient
- Chill electronic
- Instrumental
