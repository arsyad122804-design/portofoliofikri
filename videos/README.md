# Folder Videos (Video Kenangan)

Masukkan video-video kenangan ke dalam folder ini.

## File yang diperlukan:
- `video1.mp4` sampai `video4.mp4` (atau lebih)

## Spesifikasi:
- Format: MP4 (H.264 codec recommended)
- Resolusi: 1920x1080 (Full HD) atau 1280x720 (HD)
- Durasi: Maksimal 5 menit per video
- Ukuran file: Maksimal 50MB per video untuk loading yang cepat

## Tips:
- Gunakan format MP4 untuk kompatibilitas terbaik
- Compress video jika ukuran terlalu besar
- Bisa menambah lebih banyak video dengan menambahkan video5.mp4, video6.mp4, dst.
- Edit di index.html untuk menambah/mengurangi video
- Video akan memiliki kontrol play/pause/volume
